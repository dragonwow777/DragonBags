DragonBags (full fork of AdiBags)
=================================
This is a full fork of AdiBags with identifiers renamed to avoid collisions with the original addon.

Key changes:
- All occurrences of "AdiBags" -> "DragonBags"
- Saved variables: AdiBagsDB -> DragonBagsDB
- Cache token: AdiBagsCache -> DragonBagsCache (if present)
- .toc Title/Notes/SavedVariables updated to DragonBags

Notes:
- You can load both AdiBags and DragonBags simultaneously.
- To trim features, edit DragonBags.toc and comment/remove modules or locales.
